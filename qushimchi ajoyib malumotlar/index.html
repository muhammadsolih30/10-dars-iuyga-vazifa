Asosiy JavaScript Buyruqlari va Tushunchalari

Bu hujjatda JavaScript dasturlash tilining eng muhim buyruqlari va kontseptsiyalari jamlangan.

1. O'zgaruvchilar (Variables)

O'zgaruvchilar ma'lumotlarni saqlash uchun ishlatiladi.

Kalit so'z Ta'rifi Misol
let Qiymati o'zgarishi mumkin bo'lgan o'zgaruvchi e'lon qilish. 
let ism = "Ali";
ism = "Vali";
const Qiymati o'zgarmaydigan o'zgaruvchi e'lon qilish. 
const yosh = 30;
// yosh = 31; // Xatolik
var Eski usul. Hozirda let va const tavsiya etiladi. 
var eskiO'zgaruvchi = "eski qiymat";

2. Ma'lumot turlari (Data Types)

JavaScript-dagi asosiy ma'lumot turlari.

Tur Ta'rifi Misol
String Satr ma'lumotlar. 
let matn = "Bu bir satr.";
Number Sonlar. 
let butunSon = 10;
let kasrSon = 10.5;
Boolean Mantiqiy qiymatlar (true yoki false). 
let to'g'rimi = true;
Array Tartiblangan ma'lumotlar ro'yxati. 
let ro'yxat = ["olma", "banan"];
Object Kalit-qiymat juftliklari to'plami. 
let shaxs = {
    ism: "Ali",
    yosh: 30
};

3. Operatorlar (Operators)

Matematik va mantiqiy amallar uchun ishlatiladi.

Tur Operatorlar Misol
Arifmetik +, -, *, /, %, ** 
let natija = 10 % 3; // 1
Taqqoslash ==, ===, !=, !==, >, <, >=, <= 
let tekshiruv = 10 === "10"; // false
Mantiqiy &&, ||, ! 
let mantiq = (5 > 3) && (2 < 4); // true

4. Shartli operatorlar (Conditional Statements)

Shartlarga bog'liq kod bloklarini bajarish.

Buyruq Misol
if, else if, else 
let yosh = 20;
if (yosh > 18) {
    console.log("Kattalar uchun");
} else {
    console.log("Yoshlar uchun");
}
switch 
let rang = "qizil";
switch (rang) {
    case "qizil":
        console.log("To'xtash");
        break;
    default:
        console.log("Boshqa rang");
}

5. Takrorlash sikllari (Loops)

Bir xil kodni bir necha marta takrorlash.

Sikl turi Misol
for sikli 
for (let i = 0; i < 3; i++) {
    console.log(i);
}
while sikli 
let i = 0;
while (i < 3) {
    console.log(i);
    i++;
}

6. Funksiyalar (Functions)

Qayta ishlatiladigan kod bloklari.

Funksiya turi Misol
Oddiy funksiya 
function salomBer(ism) {
    console.log("Salom, " + ism);
}
salomBer("Ali");
Arrow funksiya 
const kvadrat = (son) => son * son;
console.log(kvadrat(5));
